# Binance Futures Trading Bot (Testnet)

## Overview
A CLI-based trading bot built in Python for placing orders on Binance USDT-M Futures Testnet.
The bot supports Market and Limit orders with proper validation, error handling, and structured logging.

## Features
- Market Orders (BUY / SELL)
- Limit Orders (BUY / SELL)
- Command-line interface (CLI)
- Input validation
- Structured logging to file
- Modular and clean codebase
